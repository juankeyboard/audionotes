## Requisitos vigentes (2024–2025)

- targetSdkVersion mínimo para Google Play: **API 34** (desde agosto 2024)
- Nuevo formato preferido para Play Store: **AAB** (Android App Bundle)
- `uses-cleartext-traffic`: debe ser `false` para apps que solo usan HTTPS
