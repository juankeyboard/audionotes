# F4 — Canales de Contacto y Soporte — Audionotes

---

## 1. Canales Oficiales

- **GitHub Issues:** Canal principal para reportar bugs y solicitar funcionalidades.
  - `github.com/juankeyboard/audionotes/issues`
- **GitHub Discussions:** Para preguntas generales, ideas y feedback de la comunidad.
  - `github.com/juankeyboard/audionotes/discussions`

---

## 2. Contribuciones

- **Pull Requests:** Cualquier persona puede contribuir código siguiendo las guías del repositorio (`CONTRIBUTING.md`).
- **README.md:** Incluirá instrucciones claras de instalación del APK y configuración de la API key.

---

## 3. Reporte de Errores

Al no usar Crashlytics ni servicios de analítica, los errores se reportan manualmente:
- El usuario abre un Issue en GitHub con los logs del error (si los tiene).
- La app mostrará mensajes de error descriptivos para facilitar el reporte.

---

## 4. Comunicación del Proyecto

Por ser un proyecto open source personal, no se contemplan redes sociales ni canales de marketing en esta etapa. La comunidad crece orgánicamente a través de GitHub.
