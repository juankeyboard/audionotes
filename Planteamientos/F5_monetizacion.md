# F5 — Modelo de Sostenibilidad — Audionotes

---

## 1. Modelo Económico

**Audionotes es completamente gratuito y de código abierto.**

No hay planes de pago, suscripciones, ni funcionalidades bloqueadas. El proyecto no genera ingresos directos.

---

## 2. Costo Real para el Usuario

El único gasto posible es el uso de la API de Gemini, que el usuario gestiona directamente:

| Uso | Costo estimado |
|---|---|
| Uso personal normal (< 1,500 req/día) | **$0** — capa gratuita de Google AI Studio |
| Uso intensivo | Tarifas de Google AI Studio (pagadas directamente a Google, no a Audionotes) |

El usuario tiene control total sobre su consumo desde su consola de Google AI Studio.

---

## 3. Sostenibilidad del Proyecto

Al ser open source, el proyecto puede sostenerse mediante:
- **Contribuciones de la comunidad** (código, documentación, traducciones).
- **GitHub Sponsors** (opcional, si el proyecto crece).
- **Forks y colaboraciones** de otros desarrolladores.

---

## 4. Distribución

- **Canal:** GitHub Releases (`github.com/juankeyboard/audionotes/releases`)
- **Formato:** APK firmado, instalable en cualquier Android (con instalación de fuentes desconocidas habilitada).
- **Sin Google Play Store** en la Fase 1.
